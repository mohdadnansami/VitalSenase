VitalSense – Real-Time Health Monitoring System

A full-stack web application that simulates real-time health monitoring using WebSockets, Node.js, and an interactive frontend dashboard.

 Project Overview

VitalSense is a real-time health monitoring system that displays live health vitals such as:
	•	Heart Rate
	•	SpO₂
	•	Temperature
	•	GSR
	•	Movement

The system uses a Node.js backend with WebSocket communication to stream live data to a responsive frontend dashboard.

 Features
